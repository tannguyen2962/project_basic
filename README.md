# Đồ án cơ bản Front-end

Là một đồ án HTML,CSS của học viên tại Kmin Academy

## Tính năng nổi bật

- Carousel
- Search Bar
- Slider với các Components
- Tin tức sự kiện
- Trang thông tin chi tiết phim
- Chi tiết đặt vé - thông tin phòng vé
- Trang search result
- User Login Page
- User Profile
- Và nhiều Tính năng nổi bật khác ...
